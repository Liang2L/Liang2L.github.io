Hexo博客迁移

发表于 2019-08-13 | 分类于 Hexo | 阅读次数: 
背景：
将Hexo环境源码上传至github后，在新电脑（什么环境都未安装）上如何将Hexo环境复刻下来，重新开始写博客上传

一、组件安装

1、git

2、nodejs

3、npm

4、hexo-cli

二、环境复刻

1、将github上hexo源码分支clone至本地（我的分支名为hexo）

2、依赖安装

== 备注 ==：在npm安装依赖时，可能会报错

此时将openssl更新即可

3、安装git推送环境

4、在github上settings中创建SSH，将获取的公钥复制进去即可

三、新环境上更新博客

1、将环境源码推送至hexo分支上

2、将生成的html静态文件推送至master分支上
