Linux常用命令

发表于 2019-07-15 | 分类于 Linux > 常用命令 | 阅读次数:

Linux

software

vim

查找字符串

/string enter N 下一个 n上一个

替换字符串

:s/vivian/sky/ 替换当前行第一个 vivian 为 sky

:s/vivian/sky/g 替换当前行所有 vivian 为 sky

:n,$s/vivian/sky/ 替换第 n 行开始到最后一行中每一行的第一个 vivian 为 sky

:n,$s/vivian/sky/g 替换第 n 行开始到最后一行中每一行所有 vivian 为 sky

n 为数字，若 n 为 .，表示从当前行开始到最后一行

:%s/vivian/sky/(等同于 :g/vivian/s//sky/) 替换每一行的第一个 vivian 为 sky

:%s/vivian/sky/g(等同于 :g/vivian/s//sky/g) 替换每一行中所有 vivian 为 sky

删除vim中的^M

:%s/^M$//g

system

通过进程号，找到软件安装目录

ls -l /proc/xxxx/cwd （XXXXX代表进程号）

防火墙状态

systemctl status firewalld.service

stop 关闭，start开启

端口号占用

netstat -nltp | grep

执行rpm包

rpm -ivh ucenter.rpm –force

每个文件夹大小

du -hls *

*可以替换成任意目录，即为查看该目录下所有文件夹占用大小

导入

导出

grant all privileges on . to rooact@’172.31.131.31’ identified by ‘Yhsj_idc@2019’;

flush privileges;

shell
